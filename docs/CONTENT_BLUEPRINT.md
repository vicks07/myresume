## Rewrite Pattern For Experience

Each highlight should be written like this:

- situation or problem
- action taken
- business or technical result

Example transformation:

Before:
"Developed a data-driven dashboard to analyze API crashes."

After:
"Built an API crash analysis dashboard that gave backend teams clearer failure patterns and faster issue diagnosis."
